Thank you for downloading Deputy Ryan's LiquidFX!

We appreciate your interest in enhancing the liquid effects in GTA V. This mod refines the realism and visual appeal of various liquids, including water, blood, oil, petrol, and mud, by adjusting parameters such as duration, color tint, and splash effects.

Installation Instructions:

Navigate to the file path: GTAV/mods/update/update.rpf/common/data/effects.
Replace the existing liquid effect files with the ones provided in this mod.
Additional Information:

This mod improves liquid effect realism by enhancing how these liquids interact visually in the game.
